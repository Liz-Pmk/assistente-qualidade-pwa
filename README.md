# BlzAgora

Aplicativo React Native (Expo) criado do zero a partir do briefing fornecido. Todas as telas descritas foram codificadas em `App.js`, utilizando React Navigation para gerenciar o fluxo entre splash, autenticação, onboarding de localização, busca de profissionais e agendamento completo com pesquisa de satisfação e chatbot interno.

## Scripts disponíveis

```bash
npm install
npm run start
npm run android
npm run ios
npm run web
```

> **Observação:** os comandos de instalação podem falhar neste ambiente por restrições de rede externas ao projeto. Basta executá-los novamente em um ambiente com acesso ao registro do npm.
