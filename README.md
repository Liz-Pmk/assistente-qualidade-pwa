# assistente-qualidade-pwa
PWA desenvolvido para a disciplina UX da FGV: Assistente de Qualidade Omnichannel para indústrias reguladas (ANVISA/MAPA). Inclui protótipo mobile-first, chatbot educacional, checklists, fichas técnicas e UX completo com 8 telas, baseado em pesquisa de mercado e fontes oficiais.
